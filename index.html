Javascript

JavaScript is a high-level, versatile, and widely-used programming language primarily known for its role in web development.

Typescript

TypeScript, a Microsoft-developed superset of JavaScript, introduces static typing for enhanced error detection, aiding in building scalable and maintainable applications.

NODE JS

Node.js is a JavaScript runtime built on Chrome's V8 engine, facilitating server-side scripting and scalable network applications with efficiency.

Express JS

Express.js is a minimal and flexible Node.js web application framework, simplifying the development of robust and scalable server-side applications.

React JS

React.js is a JavaScript library for building user interfaces, providing a declarative and efficient way to create interactive and dynamic applications.

Next JS

Next.js is a React framework for building server-rendered and static web applications, providing a simple and intuitive developer experience.

MongoDB

MongoDB is a NoSQL database, storing data in flexible, JSON-like documents. It's known for scalability, flexibility, and real-time data capabilities.

PostgreSQL

PostgreSQL is a powerful open-source relational database system. It supports SQL queries, ACID transactions, and extensibility, ideal for various applications.

TailwindCSS

Tailwind CSS is a utility-first CSS framework for building modern, responsive designs with pre-defined classes, enhancing rapid development efficiency.

Amazon Web services

Amazon Web Services (AWS) is a cloud computing platform offering a vast array of services, including computing power, storage, and databases.

Figma

Figma is a collaborative design platform, enabling teams to create, share, and prototype user interfaces with real-time collaboration and feedback.
